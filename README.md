# Homebrew Tap for Trust Me Bro (TMB)

```bash
brew tap ZaxShen/tmb
brew install trustmybot
```

After install, use any of these commands:

```bash
bro              # Chat with your bro
bro plan         # Full planning workflow
bro --help       # All commands
```

Aliases: `bot` and `tmb` also work.

## Upgrade

```bash
brew upgrade trustmybot
```
